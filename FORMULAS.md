# Документация по формулам SmartTable

## Обзор

SmartTable поддерживает широкий спектр формул для работы с данными в таблицах. Все формулы начинаются со знака `=`.

## Математические формулы

### SUM
**Описание:** Вычисляет сумму значений.

**Синтаксис:** `=SUM(диапазон)`

**Примеры:**
```
=SUM(A1:A10)          // Сумма значений от A1 до A10
=SUM(A1, B1, C1)      // Сумма отдельных ячеек
=SUM(A1:A5, C1:C5)    // Сумма нескольких диапазонов
```

### AVERAGE
**Описание:** Вычисляет среднее арифметическое значений.

**Синтаксис:** `=AVERAGE(диапазон)`

**Примеры:**
```
=AVERAGE(A1:A10)      // Среднее значение от A1 до A10
=AVERAGE(B1:B5, D1:D5)
```

### MIN
**Описание:** Возвращает минимальное значение.

**Синтаксис:** `=MIN(диапазон)`

**Примеры:**
```
=MIN(A1:A100)         // Минимальное значение в диапазоне
```

### MAX
**Описание:** Возвращает максимальное значение.

**Синтаксис:** `=MAX(диапазон)`

**Примеры:**
```
=MAX(A1:A100)         // Максимальное значение в диапазоне
```

### COUNT
**Описание:** Подсчитывает количество числовых значений.

**Синтаксис:** `=COUNT(диапазон)`

**Примеры:**
```
=COUNT(A1:A10)        // Количество чисел в диапазоне
```

### COUNTA
**Описание:** Подсчитывает количество непустых ячеек.

**Синтаксис:** `=COUNTA(диапазон)`

**Примеры:**
```
=COUNTA(A1:A10)       // Количество заполненных ячеек
```

### PRODUCT
**Описание:** Вычисляет произведение значений.

**Синтаксис:** `=PRODUCT(диапазон)`

**Примеры:**
```
=PRODUCT(A1:A5)       // Произведение значений
=PRODUCT(A1, B1, C1)
```

### ROUND
**Описание:** Округляет число до указанного количества знаков.

**Синтаксис:** `=ROUND(число, количество_знаков)`

**Примеры:**
```
=ROUND(A1, 2)         // Округлить до 2 знаков
=ROUND(3.14159, 2)    // Результат: 3.14
```

### ROUNDUP
**Описание:** Округляет число вверх (по модулю).

**Синтаксис:** `=ROUNDUP(число, количество_знаков)`

**Примеры:**
```
=ROUNDUP(3.141, 2)    // Результат: 3.15
```

### ROUNDDOWN
**Описание:** Округляет число вниз (по модулю).

**Синтаксис:** `=ROUNDDOWN(число, количество_знаков)`

**Примеры:**
```
=ROUNDDOWN(3.141, 2)  // Результат: 3.14
```

### ABS
**Описание:** Возвращает модуль числа.

**Синтаксис:** `=ABS(число)`

**Примеры:**
```
=ABS(-5)              // Результат: 5
=ABS(A1)
```

### POWER
**Описание:** Возводит число в степень.

**Синтаксис:** `=POWER(число, степень)`

**Примеры:**
```
=POWER(2, 3)          // Результат: 8 (2³)
=POWER(A1, 2)         // Квадрат значения A1
```

### SQRT
**Описание:** Возвращает квадратный корень.

**Синтаксис:** `=SQRT(число)`

**Примеры:**
```
=SQRT(16)             // Результат: 4
=SQRT(A1)
```

### MOD
**Описание:** Возвращает остаток от деления.

**Синтаксис:** `=MOD(число, делитель)`

**Примеры:**
```
=MOD(10, 3)           // Результат: 1
=MOD(A1, 2)           // Проверка на чётность
```

## Логические формулы

### IF
**Описание:** Проверяет условие и возвращает одно значение если истинно, другое если ложно.

**Синтаксис:** `=IF(условие, значение_если_истина, значение_если_ложь)`

**Поддерживаемые операторы сравнения:**
- `=` — равно
- `<>` или `!=` — не равно
- `>` — больше
- `<` — меньше
- `>=` — больше или равно
- `<=` — меньше или равно

**Примеры:**
```
=IF(A1>10, "Больше", "Меньше")
=IF(A1=B1, "Равны", "Не равны")
=IF(A1>=50, "Зачёт", "Не зачёт")
=IF(A1>100, A1*0.9, A1)  // Скидка 10% если больше 100
```

### IFS
**Описание:** Проверяет несколько условий и возвращает значение для первого истинного условия.

**Синтаксис:** `=IFS(условие1, значение1, условие2, значение2, ..., TRUE, значение_по_умолчанию)`

**Примеры:**
```
=IFS(A1>90, "Отлично", A1>70, "Хорошо", A1>50, "Удовл.", TRUE, "Плохо")
=IFS(A1=1, "Красный", A1=2, "Зелёный", A1=3, "Синий", TRUE, "Другой")
```

### AND
**Описание:** Возвращает ИСТИНА если все условия истинны.

**Синтаксис:** `=AND(условие1, условие2, ...)`

**Примеры:**
```
=AND(A1>0, A1<100)      // ИСТИНА если от 0 до 100
=IF(AND(A1>0, A1<100), "В диапазоне", "Вне диапазона")
```

### OR
**Описание:** Возвращает ИСТИНА если хотя бы одно условие истинно.

**Синтаксис:** `=OR(условие1, условие2, ...)`

**Примеры:**
```
=OR(A1=1, A1=2, A1=3)   // ИСТИНА если 1, 2 или 3
=IF(OR(A1<0, A1>100), "Вне диапазона", "В диапазоне")
```

### NOT
**Описание:** Инвертирует логическое значение.

**Синтаксис:** `=NOT(условие)`

**Примеры:**
```
=NOT(A1>10)             // ИСТИНА если A1 <= 10
=IF(NOT(A1=""), "Не пусто", "Пусто")
```

### IFERROR
**Описание:** Возвращает альтернативное значение если формула вызывает ошибку.

**Синтаксис:** `=IFERROR(формула, значение_при_ошибке)`

**Примеры:**
```
=IFERROR(A1/B1, 0)      // 0 при делении на ноль
=IFERROR(VLOOKUP(A1, B:C, 2, FALSE), "Не найдено")
```

### TRUE
**Описание:** Возвращает логическое значение ИСТИНА.

**Синтаксис:** `=TRUE()`

**Примеры:**
```
=TRUE()                 // ИСТИНА
```

### FALSE
**Описание:** Возвращает логическое значение ЛОЖЬ.

**Синтаксис:** `=FALSE()`

**Примеры:**
```
=FALSE()                // ЛОЖЬ
```

## Текстовые формулы

### CONCATENATE / CONCAT
**Описание:** Сцепляет несколько текстовых строк в одну.

**Синтаксис:** `=CONCATENATE(текст1, текст2, ...)` или `=CONCAT(текст1, текст2, ...)`

**Примеры:**
```
=CONCATENATE(A1, " ", B1)     // "Имя Фамилия"
=CONCAT(A1, B1, C1)
=A1 & " " & B1                // Альтернативный синтаксис
```

### LEN
**Описание:** Возвращает количество символов в тексте.

**Синтаксис:** `=LEN(текст)`

**Примеры:**
```
=LEN(A1)                      // Длина текста в A1
=LEN("Привет")                // Результат: 6
```

### UPPER
**Описание:** Преобразует текст в верхний регистр.

**Синтаксис:** `=UPPER(текст)`

**Примеры:**
```
=UPPER(A1)                    // "ПРИВЕТ"
=UPPER("привет")              // Результат: "ПРИВЕТ"
```

### LOWER
**Описание:** Преобразует текст в нижний регистр.

**Синтаксис:** `=LOWER(текст)`

**Примеры:**
```
=LOWER(A1)                    // "привет"
=LOWER("ПРИВЕТ")              // Результат: "привет"
```

### PROPER
**Описание:** Делает заглавными первые буквы каждого слова.

**Синтаксис:** `=PROPER(текст)`

**Примеры:**
```
=PROPER(A1)                   // "Привет Мир"
=PROPER("привет мир")         // Результат: "Привет Мир"
```

### TRIM
**Описание:** Удаляет лишние пробелы из текста.

**Синтаксис:** `=TRIM(текст)`

**Примеры:**
```
=TRIM(A1)                     // Удаляет лишние пробелы
=TRIM("  Привет  ")           // Результат: "Привет"
```

### LEFT
**Описание:** Возвращает указанное количество символов с начала строки.

**Синтаксис:** `=LEFT(текст, количество)`

**Примеры:**
```
=LEFT(A1, 3)                  // Первые 3 символа
=LEFT("Привет", 3)            // Результат: "При"
```

### RIGHT
**Описание:** Возвращает указанное количество символов с конца строки.

**Синтаксис:** `=RIGHT(текст, количество)`

**Примеры:**
```
=RIGHT(A1, 3)                 // Последние 3 символа
=RIGHT("Привет", 3)           // Результат: "вет"
```

### MID
**Описание:** Возвращает указанное количество символов из середины строки.

**Синтаксис:** `=MID(текст, начальная_позиция, количество)`

**Примеры:**
```
=MID(A1, 2, 3)                // 3 символа начиная со 2-го
=MID("Привет", 2, 3)          // Результат: "рив"
```

### SUBSTITUTE
**Описание:** Заменяет один текст другим.

**Синтаксис:** `=SUBSTITUTE(текст, старый_текст, новый_текст)`

**Примеры:**
```
=SUBSTITUTE(A1, "старый", "новый")
=SUBSTITUTE("Привет мир", "мир", "пока")  // Результат: "Привет пока"
```

### TEXT
**Описание:** Форматирует число в текст по указанному формату.

**Синтаксис:** `=TEXT(число, формат)`

**Примеры:**
```
=TEXT(A1, "0.00")             // "3.14"
=TEXT(A1, "#,##0")            // "1 234"
=TEXT(0.5, "0%")              // "50%"
```

### FIND
**Описание:** Находит позицию текста (с учётом регистра).

**Синтаксис:** `=FIND(искомый_текст, где_искать)`

**Примеры:**
```
=FIND("р", "Привет")          // Результат: 3
=FIND("П", "Привет")          // Результат: 1
```

### SEARCH
**Описание:** Находит позицию текста (без учёта регистра).

**Синтаксис:** `=SEARCH(искомый_текст, где_искать)`

**Примеры:**
```
=SEARCH("п", "Привет")        // Результат: 1
=SEARCH("мир", "Привет мир")  // Результат: 8
```

## Формулы даты и времени

### TODAY
**Описание:** Возвращает текущую дату.

**Синтаксис:** `=TODAY()`

**Примеры:**
```
=TODAY()                      // Текущая дата
```

### NOW
**Описание:** Возвращает текущую дату и время.

**Синтаксис:** `=NOW()`

**Примеры:**
```
=NOW()                        // Текущая дата и время
```

### DAY
**Описание:** Возвращает день месяца (1-31).

**Синтаксис:** `=DAY(дата)`

**Примеры:**
```
=DAY(A1)                      // День из даты в A1
=DAY("15.01.2024")            // Результат: 15
```

### MONTH
**Описание:** Возвращает месяц (1-12).

**Синтаксис:** `=MONTH(дата)`

**Примеры:**
```
=MONTH(A1)                    // Месяц из даты в A1
=MONTH("15.01.2024")          // Результат: 1
```

### YEAR
**Описание:** Возвращает год.

**Синтаксис:** `=YEAR(дата)`

**Примеры:**
```
=YEAR(A1)                     // Год из даты в A1
=YEAR("15.01.2024")           // Результат: 2024
```

### HOUR
**Описание:** Возвращает часы (0-23).

**Синтаксис:** `=HOUR(время)`

**Примеры:**
```
=HOUR(A1)                     // Часы из времени в A1
```

### MINUTE
**Описание:** Возвращает минуты (0-59).

**Синтаксис:** `=MINUTE(время)`

**Примеры:**
```
=MINUTE(A1)                   // Минуты из времени в A1
```

### SECOND
**Описание:** Возвращает секунды (0-59).

**Синтаксис:** `=SECOND(время)`

**Примеры:**
```
=SECOND(A1)                   // Секунды из времени в A1
```

### DATE
**Описание:** Создаёт дату из года, месяца и дня.

**Синтаксис:** `=DATE(год, месяц, день)`

**Примеры:**
```
=DATE(2024, 1, 15)            // 15.01.2024
=DATE(YEAR(A1), MONTH(A1)+1, DAY(A1))  // Следующий месяц
```

### TIME
**Описание:** Создаёт время из часов, минут и секунд.

**Синтаксис:** `=TIME(часы, минуты, секунды)`

**Примеры:**
```
=TIME(12, 30, 0)              // 12:30:00
```

## Формулы поиска и ссылок

### VLOOKUP
**Описание:** Вертикальный поиск значения в таблице.

**Синтаксис:** `=VLOOKUP(искомое_значение, таблица, номер_столбца, точное_совпадение)`

**Примеры:**
```
=VLOOKUP(A1, B1:C10, 2, FALSE)    // Точное совпадение
=VLOOKUP(A1, B1:C10, 2, TRUE)     // Приблизительное совпадение
```

### HLOOKUP
**Описание:** Горизонтальный поиск значения в таблице.

**Синтаксис:** `=HLOOKUP(искомое_значение, таблица, номер_строки, точное_совпадение)`

**Примеры:**
```
=HLOOKUP(A1, B1:F2, 2, FALSE)     // Точное совпадение
```

### INDEX
**Описание:** Возвращает значение по индексу из диапазона.

**Синтаксис:** `=INDEX(диапазон, номер_строки, номер_столбца)`

**Примеры:**
```
=INDEX(A1:C10, 2, 3)              // Значение в строке 2, столбце 3
```

### MATCH
**Описание:** Возвращает позицию значения в диапазоне.

**Синтаксис:** `=MATCH(искомое_значение, диапазон, тип_совпадения)`

**Параметры:**
- `0` — точное совпадение
- `1` — приближённое (меньше или равно)
- `-1` — приближённое (больше или равно)

**Примеры:**
```
=MATCH(A1, B1:B10, 0)             // Позиция точного совпадения
```

## Финансовые формулы

### PMT
**Описание:** Вычисляет размер периодического платежа по кредиту.

**Синтаксис:** `=PMT(ставка, количество_периодов, сумма_кредита)`

**Примеры:**
```
=PMT(0.05/12, 60, 10000)          // Ежемесячный платёж
```

### FV
**Описание:** Вычисляет будущую стоимость инвестиции.

**Синтаксис:** `=FV(ставка, количество_периодов, платёж)`

**Примеры:**
```
=FV(0.05/12, 60, -100)            // Будущая стоимость
```

### PV
**Описание:** Вычисляет текущую стоимость инвестиции.

**Синтаксис:** `=PV(ставка, количество_периодов, платёж)`

**Примеры:**
```
=PV(0.05/12, 60, -100)            // Текущая стоимость
```

## Операторы сравнения

| Оператор | Описание | Пример |
|----------|----------|--------|
| `=` | Равно | `A1=B1` |
| `<>` | Не равно | `A1<>B1` |
| `!=` | Не равно (альтернатива) | `A1!=B1` |
| `>` | Больше | `A1>10` |
| `<` | Меньше | `A1<10` |
| `>=` | Больше или равно | `A1>=10` |
| `<=` | Меньше или равно | `A1<=10` |

## Примеры использования

### Базовые вычисления
```
=A1+B1              // Сложение
=A1-B1              // Вычитание
=A1*B1              // Умножение
=A1/B1              // Деление
=A1^2               // Возведение в степень
```

### Сложные условия
```
=IF(AND(A1>0, A1<100), "В диапазоне", "Вне диапазона")
=IF(OR(A1<0, A1>100), "Ошибка", "OK")
=IF(A1>=90, "A", IF(A1>=70, "B", IF(A1>=50, "C", "F")))
```

### Работа с текстом
```
=UPPER(LEFT(A1, 1)) & LOWER(RIGHT(A1, LEN(A1)-1))  // Заглавная первая буква
=TRIM(CONCATENATE(A1, " ", B1))                     // Полное имя
```

### Финансовые расчёты
```
=IF(A1>1000, A1*0.9, A1)                            // Скидка 10%
=A1*B1*(1-C1)                                       // Цена со скидкой
```

### Даты
```
=YEAR(TODAY())                                      // Текущий год
=DATE(YEAR(A1)+1, MONTH(A1), DAY(A1))              // Через год
```

## Ссылки на ячейки

### Типы ссылок

| Тип | Пример | Описание |
|-----|--------|----------|
| Относительная | `A1` | Изменяется при копировании |
| Абсолютная | `$A$1` | Не изменяется при копировании |
| Смешанная | `$A1` или `A$1` | Частично фиксируется |

### Диапазоны
```
A1:A10            // Столбец A, строки 1-10
A1:E1             // Строка 1, столбцы A-E
A1:E10            // Прямоугольный диапазон
```

## Обработка ошибок

| Ошибка | Описание |
|--------|----------|
| `#NAME?` | Неизвестная функция |
| `#ERROR!` | Ошибка в формуле |
| `#N/A` | Значение не найдено |
| `#VALUE!` | Неверный тип аргумента |
| `#DIV/0!` | Деление на ноль |

### Примеры обработки ошибок
```
=IFERROR(A1/B1, 0)                    // 0 при ошибке
=IFERROR(VLOOKUP(A1, B:C, 2, 0), "Не найдено")
=IF(ISERROR(A1/B1), "Ошибка", A1/B1)
```

## Советы и рекомендации

1. **Автодополнение:** При вводе `=` появляется список функций
2. **Навигация:** Используйте стрелки для выбора функции в списке
3. **Tab/Enter:** Вставляет выбранную функцию
4. **Esc:** Закрывает подсказку

### Оптимизация
- Избегайте циклических ссылок
- Используйте диапазоны вместо отдельных ячеек
- Кэшируйте результаты сложных вычислений
