# AutoComplete — Автозаполнение ячеек

## Обзор

Система автозаполнения SmartTable предоставляет умные подсказки при вводе данных в ячейки. Работает полностью локально, без внешних API и интернет-соединения.

## Возможности

### 1. Автодополнение формул

При вводе `=` система предлагает формулы:

```
=S   → =SUM()
=A   → =AVERAGE()
=I   → =IF(, , )
=V   → =VLOOKUP(, , , FALSE)
```

### 2. Числовые последовательности

Автоматически распознаёт паттерны:

```
2, 4, 6, ?  → 8 (арифметическая прогрессия)
2, 4, 8, ?  → 16 (геометрическая прогрессия)
1, 2, 3, ?  → 4 (линейный тренд)
```

### 3. Текстовые паттерны

Распознаёт текстовые последовательности:

```
Item 1
Item 2
Item 3
? → Item 4
```

## Использование

### Ввод формул

1. Начните вводить `=` в ячейке
2. Появится подсказка с формулой
3. Нажмите **Tab** или **Enter** для принятия

### Числовое автозаполнение

1. Введите несколько чисел в колонку
2. В следующей ячейке начните вводить первое число
3. Система предложит продолжение последовательности

### Принятие подсказки

- **Tab** — принять подсказку
- **Enter** — принять подсказку (без перехода на новую строку)
- **Escape** — скрыть подсказку

## API

### Глобальный объект

```javascript
window.autoComplete
```

### Методы

#### `init()`
Инициализация системы (вызывается автоматически при загрузке).

#### `getPrediction(row, col)`
Получить предсказание для ячейки.

```javascript
const prediction = window.autoComplete.getPrediction(0, 0);
console.log(prediction);
// { type: 'formula', completion: '=SUM()', confidence: 0.95 }
```

#### `accept()`
Принять текущую подсказку.

```javascript
window.autoComplete.accept();
```

#### `clearCache()`
Очистить кэш значений.

```javascript
window.autoComplete.clearCache();
```

### Конфигурация

```javascript
window.autoComplete.config = {
  suggestionDelay: 150,        // Задержка перед показом (мс)
  minCharsForSuggestion: 1,    // Мин. символов для подсказки
  placeholderColor: '#a0a0a0', // Цвет серого текста
  maxSuggestions: 5,           // Макс. количество подсказок
  confidenceThreshold: 0.6     // Порог уверенности
};
```

## Поддерживаемые формулы

### Математические
- `SUM` — сумма
- `AVERAGE` — среднее
- `MIN` — минимум
- `MAX` — максимум
- `COUNT` — количество

### Логические
- `IF` — условие
- `IFS` — несколько условий
- `AND` — И
- `OR` — ИЛИ

### Текстовые
- `CONCATENATE` — сцепить текст
- `LEN` — длина
- `UPPER` — верхний регистр
- `LOWER` — нижний регистр

### Поиск
- `VLOOKUP` — вертикальный поиск
- `HLOOKUP` — горизонтальный поиск
- `INDEX` — значение по индексу
- `MATCH` — позиция значения

## Алгоритмы

### Анализ числовых последовательностей

1. **Арифметическая прогрессия**
   - Проверяет постоянную разницу
   - Уверенность: 90%

2. **Геометрическая прогрессия**
   - Проверяет постоянное отношение
   - Уверенность: 85%

3. **Линейный тренд**
   - Простая линейная регрессия
   - Уверенность: 70%

### Анализ текстовых паттернов

1. **Числовые суффиксы**
   - Распознаёт "Item 1, Item 2..."
   - Уверенность: 85%

## Интеграция с Focus Manager

После принятия подсказки фокус автоматически восстанавливается на активной ячейке.

```javascript
// В autocomplete.js
acceptSuggestion(suggestion, cell) {
  // ...
  if (typeof window.restoreFocus === 'function') {
    window.restoreFocus();
  }
}
```

## Производительность

- **Задержка**: 150мс перед показом подсказки
- **Кэширование**: значения ячеек кэшируются в Map
- **Оптимизация**: подсказки скрываются при скролле

## Тёмная тема

  border-color: #444;

## Расширение

### Добавление новой формулы

Откройте `autocomplete.js` и добавьте в `FORMULA_PATTERNS`:

```javascript
'NEWFORMULA': {
  pattern: /^=NF/i,
  completion: '=NEWFORMULA()',
  cursorOffset: -1,
  description: 'Описание формулы',
  example: '=NEWFORMULA(A1, B1)'
}
```

### Добавление нового паттерна

Добавьте функцию анализа в `analyzeNumericPattern` или `analyzeTextPattern`.

## Ограничения

- Максимум 100 строк для анализа
- Задержка 150мс для производительности
- Не работает в режиме редактирования нескольких ячеек
