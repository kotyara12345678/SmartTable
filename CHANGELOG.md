# Changelog

Все изменения в проекте SmartTable.

## [2026-03-08]

### Исправлено
- **Потеря фокуса** — добавлен Focus Manager для восстановления фокуса после операций
- **Очистка кеша** — фокус теперь восстанавливается после `clearHistory()` и `resetTableState()`
- **Прокрутка колонок** — исправлено "наезжание" колонок при изменении размера
- **Confirm блокировка** — убран `confirm()` из `openDocument()` (2175ms → <50ms)

### Изменения
- **Формулы IF/ELSE** — полная поддержка операторов сравнения (`=`, `<>`, `!=`, `>`, `<`, `>=`, `<=`)
- **Формула IFS** — добавлена поддержка нескольких условий
- **SUM диапазоны** — исправлено разворачивание массивов в формулах

### Рефакторинг
- **Renderer модули** — разбит `renderer.ts` (7000 строк) на модули:
  - `cell-rendering.ts` — отрисовка ячеек
  - `selection.ts` — выделение и навигация
  - `editing.ts` — редактирование ячеек
  - `scroll.ts` — прокрутка и виртуализация
  - `formatting.ts` — форматирование текста
  - `borders.ts` — границы ячеек
  - `column-row.ts` — управление колонками/строками

### Документация
- `FORMULAS.md` — полная документация по формулам
- `modules/README.md` — документация по модулям renderer
- `BUILD.md` — инструкция по сборке и запуску

### Новые формулы
- `MOD` — остаток от деления
- `ROUNDUP` — округление вверх
- `ROUNDDOWN` — округление вниз
- `PROPER` — заглавные буквы каждого слова
- `TEXT` — форматирование числа
- `FIND` — найти текст (с учётом регистра)
- `SEARCH` — найти текст (без учёта регистра)
- `DATE` — дата из компонентов
- `TIME` — время из компонентов
- `TRUE` — логическая ИСТИНА
- `FALSE` — логическая ЛОЖЬ

## [2026-03-07]

### Добавлено
- Focus Manager с функциями:
  - `saveActiveCell()` — сохранить активную ячейку
  - `restoreFocus()` — восстановить фокус
  - `onAfterFileOperation()` — вызов после операций с файлом

### Исправлено
- Импорты в Electron — добавлено расширение `.js` в импортах ES-модулей

## [2026-03-06]

### Добавлено
- Система плагинов
- AI помощник для работы с таблицами
- Авто сохранение каждые 1 минуту

### Изменения
- Улучшена работа с локальной базой данных
- Оптимизирована виртуальная прокрутка
