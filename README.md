# SmartTable 🚀

**SmartTable** — это современное кроссплатформенное приложение для работы с электронными таблицами, вдохновленное Excel, с интеграцией искусственного интеллекта для автоматизации рутинных задач.

## ✨ Возможности

### 📊 Таблицы
- ✅ **100 строк × 100 столбцов** (A-YU) на лист
- ✅ **Неограниченное количество листов**
- ✅ **Выделение диапазонов** мышью
- ✅ **Контекстное меню** (ПКМ)
- ✅ **Fill Handle** для автозаполнения с распознаванием паттернов
- ✅ **Изменение размера** строк и столбцов перетаскиванием
- ✅ **Форматирование ячеек** (жирный, курсив, цвет, выравнивание)
- ✅ **Автосохранение** с настраиваемым интервалом
- ✅ **Оптимизация** — рендеринг только видимых ячеек

### 🎨 Форматирование
- ✅ Шрифты (Arial, Calibri, Times New Roman, Verdana)
- ✅ Размеры шрифта (8-16)
- ✅ Начертание (жирный, курсив, подчеркнутый, зачеркнутый)
- ✅ Цвет текста и заливки
- ✅ Выравнивание (лево, центр, право, перенос)
- ✅ Форматы чисел (общий, числовой, валюта, процент, дата)
- ✅ Границы и объединение ячеек

### 📈 Диаграммы
- ✅ **Столбчатые** (bar, horizontal bar)
- ✅ **Линейные** (line, area)
- ✅ **Круговые** (pie, doughnut)
- ✅ **Полярные** (polar area, radar, scatter, bubble)
- ✅ Экспорт диаграмм
- ✅ Создание из выделенного диапазона

### 🧮 Формулы
- ✅ **Математические**: SUM, AVERAGE, MIN, MAX, COUNT, PRODUCT, ROUND, ABS, POWER, SQRT

## 🔌 Плагины

SmartTable поддерживает систему плагинов для расширения функциональности.

### Установка плагина

1. Скачайте ZIP-архив плагина из GitHub Releases
2. Нажмите кнопку **Расширения** на верхней панели (иконка с слоями 📦)
3. Нажмите **Установить плагин**
4. Выберите ZIP-файл
5. Плагин появится в списке установленных

### Структура плагина

```
my-plugin/
├── manifest.json    # Метаданные (name, version, permissions)
├── main.js          # Точка входа
├── styles.css       # Стили (опционально)
└── assets/          # Ресурсы (иконки, изображения)
```

### Разработка плагинов

Полная документация для разработчиков доступна в файле [DEVELOPER_GUIDE.md](sheets/src/electron/plugins/DEVELOPER_GUIDE.md)

**Пример простой функции плагина:**

```javascript
// main.js
export function activate(api) {
  // Добавление кнопки в ленту
  api.ui.addRibbonButton({
    id: 'my-btn',
    groupId: 'insert',
    icon: '<svg>...</svg>',
    label: 'Моя кнопка',
    onClick: () => {
      const range = api.sheets.getSelectedRange();
      api.ui.showNotification('Выделено: ' + JSON.stringify(range));
    }
  });
}
```

### Доступные API

- **Sheets API** — работа с ячейками и листами
- **UI API** — создание кнопок, панелей, уведомлений
- **Events API** — подписка на события (изменение ячеек, открытие файлов)
- **Storage API** — хранение данных плагина

### Пример плагина

В папке `plugins/example-plugin/` находится пример плагина с базовой структурой.

## 🤖 ИИ Помощник

### Режимы работы

#### 🧑‍💼 Assistant (по умолчанию)
- Быстрые ответы на простые вопросы
- Выполнение одиночных действий
- Примеры:
  - "Покрась A1 в красный"
  - "Заполни таблицу данными"
  - "Посчитай сумму в столбце C"

#### 🤖 Agent (для сложных задач)
- Многошаговые планы выполнения
- Анализ данных и поиск закономерностей
- Создание отчетов
- Примеры:
  - "Проанализируй данные и найди топ-5"
  - "Создай отчет по продажам"
  - "Преобразуй таблицу и создай диаграмму"

### Команды ИИ

| Команда | Описание |
|---------|----------|
| `set_cell` | Установить значение ячейки |
| `set_cell_color` | Покрасить ячейку |
| `color_column` | Покрасить столбец |
| `color_row` | Покрасить строку |
| `set_formula` | Вставить формулу |
| `fill_table` | Заполнить таблицу |
| `clear_cell` | Очистить ячейку |
| `create_chart` | Создать диаграмму |

### Примеры запросов

```
📊 "Заполни таблицу данными о продажах"
🎨 "Покрась столбец A в зеленый"
📈 "Создай диаграмму по данным за 2025 год"
🔍 "Найди максимальное значение в столбце C"
📝 "Добавь формулу средней выручки"
```

- Основная область таблицы
- Заголовки строк и столбцов
- Выделение ячеек

### DashboardComponent

## 📐 Формулы

### Синтаксис

```
=SUM(A1:A10)           # Сумма значений
=AVERAGE(B1:B10)       # Среднее значение
=IF(A1>10, "Да", "Нет") # Условие
=VLOOKUP(A1, B1:C10, 2, FALSE) # Вертикальный поиск
=AI("Сумма столбцов A и B") # ИИ запрос
```

### Автокомплит

При вводе `=` появляется список доступных формул с описанием и синтаксисом.
