# Папка для картинок Splash Screen

## Как использовать

1. **Положите картинку** с именем `splash.png` в эту папку
2. **Перезапустите приложение**
3. **Готово!** Картинка будет показана при запуске

## Рекомендации

- **Формат:** PNG, JPG, BMP, GIF
- **Размер:** 600x400 пикселей (optimal)
- **Имя файла:** ОБЯЗАТЕЛЬНО `splash.png`

## Если нужна заглушка

Запустите генератор:
```bash
python generate_splash.py
```

Будет создана картинка в стиле PyCharm (требует pillow: pip install pillow).

## Примеры

### Создать свою картинку

1. Откройте GIMP, Photoshop или Paint.NET
2. Создайте 600x400 картинку
3. Нарисуйте свой дизайн (логотип, текст и т.д.)
4. Сохраните как `splash.png` в этой папке
5. Готово!

### Найти уже готовую

Ищите на:
- pixabay.com
- unsplash.com
- pexels.com

Скачайте 600x400 и сохраните как `splash.png`.

## Что будет если нет картинки?

Будет показан серый экран с текстом.

Это нормально - приложение работает, просто картинка не найдена.

---

**Главное** - положите `splash.png` с нужной картинкой и перезапустите приложение.
